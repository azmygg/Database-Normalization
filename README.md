# Database Normalization – Normal Forms (1NF → 5NF)

A presentation that explains **Database Normalization** concepts and demonstrates how higher normal forms improve database structure by reducing redundancy and preventing anomalies.

This presentation was prepared as part of the **Database Course**.

---

## 📌 Overview

Database Normalization is a process used to organize data efficiently by:

* Reducing data redundancy
* Improving data integrity
* Preventing insertion, update, and deletion anomalies
* Making databases easier to maintain

This presentation covers both foundational and advanced normalization concepts.
